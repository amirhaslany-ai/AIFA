# Brand Guidelines
**Document ID:** `System/Brand_Guidelines.md`
**Referenced by:** `05_Production/Video_Assembly.md` (lower-thirds, intro/outro), `07_SEO/README.md` (thumbnails, D-018)

> Distinct from `System/Shima_Persona.md`: that file is the on-camera character bible (what Shima looks/sounds like). This file is the **channel's** visual identity — logo, color palette, typography, and how they apply to non-persona assets (lower-thirds, thumbnails, channel art, intro/outro). Where the two overlap (e.g. a signature color that appears both on Shima's wardrobe and in lower-thirds), that value is defined once here and referenced by the persona file, not duplicated.

## Purpose

Give `05_Production` (video assembly) and `07_SEO` (thumbnails) a single, consistent visual system so every exported asset is recognizably AIFA at a glance — independent of which video, which platform, or which engine produced it.

## 🔶 NEEDS FOUNDER INPUT — core visual identity

No logo, color palette, or typography has been specified in any prior material reviewed. This is a genuine gap, not an oversight — until it's filled, `05_Production`'s brand-element step and `07_SEO`'s thumbnail work are both structurally blocked on this file (see `05_Production/Video_Assembly.md` § Open Gaps, `07_SEO/README.md` § Thumbnails).

Needed, at minimum:
- **Logo** — primary mark, and how it appears in a lower-third vs. a thumbnail vs. channel art (these are different use cases with different size/legibility constraints).
- **Color palette** — primary/secondary colors, and whether they're meant to visually connect to Shima's wardrobe/set (`System/Shima_Persona.md` §7–8, also open).
- **Typography** — for on-screen text (lower-thirds, captions styling beyond the plain captions mandated by D-015, thumbnail text overlays).

## What can be specified regardless of the gap above

- **Consistency requirement:** whatever the values above turn out to be, every platform export must use them identically — Constitution Core Principle 3's persona-consistency requirement extends to brand elements, not just Shima herself.
- **Thumbnail honesty constraint:** per `07_SEO/README.md` § Thumbnails, a thumbnail may never depict or imply something the video doesn't actually contain, regardless of what the final visual style looks like.
- **Captions are not a brand-styling decision:** D-015's caption mandate is independent of whatever typography this file eventually specifies for other on-screen text — captions default to platform-native/plain unless this file later specifies otherwise.
